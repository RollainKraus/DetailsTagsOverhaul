# EditTagsOverhaul

Hierarchical tag browser for Stash scene, gallery, and image edit pages.

## Features
- Group tags by parent hierarchy
- Optional nested subgroups
- Optional multi-parent duplication
- Text, image, or image+text tag display
- Adjustable image size
- Adjustable selected border color

## Files
- `EditTagsOverhaul.yml`
- `EditTagsOverhaul.js`
- `EditTagsOverhaul.css`

## Install
Copy the folder into your Stash `plugins` directory, then reload plugins.
